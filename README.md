# DatabaseManager_CLI
A database manager created in Java. Performs CRUD operations and has permanent storage

# Functionalities
A user can use the following functionalities:
  1. Create table
  2. Show tables
  3. Drop table
  4. Insert in table
  5. Delete from table
  6. Select from table
  7. Update from table
  8. Help (For correct syntax)
